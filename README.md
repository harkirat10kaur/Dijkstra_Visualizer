# Dijkstra Algorithm Visualizer (Link State Routing)

A GUI-based visualizer for **Dijkstra’s Algorithm**, built using **Python**.  
This project helps understand how **Link State Routing** protocols compute the shortest path in computer networks.

---

## Features
- Add custom nodes and weighted edges  
- Visualize the network graph dynamically  
- Run Dijkstra’s Algorithm with a single click  
- Highlight the shortest path graphically  
- Automatically generate routing tables  
- Simple and user-friendly GUI  

---

## Tech Stack
- **Python**
- **Tkinter** – GUI
- **NetworkX** – Graph algorithms
- **Matplotlib** – Graph visualization

---

## How It Works
1. Add nodes (e.g., A, B, C)  
2. Add edges with weights (cost)  
3. Enter source and destination nodes  
4. Click ▶ **START DIJKSTRA**  
5. View the shortest path and routing table  

---
